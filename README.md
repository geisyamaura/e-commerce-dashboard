# E-Commerce Data Analysis Project

## Deskripsi
Proyek ini bertujuan untuk menganalisis performa bisnis e-commerce berdasarkan data transaksi.

## Pertanyaan Bisnis
1. Kategori produk apa yang menghasilkan revenue terbesar?
2. Bagaimana distribusi metode pembayaran pelanggan?

## Insight
- Kategori tertentu mendominasi penjualan
- Metode pembayaran menunjukkan preferensi user
- Ada customer yang kontribusinya besar terhadap revenue

## Cara Menjalankan Dashboard
1. Masuk ke folder dashboard:
cd dashboard

2. Jalankan:
streamlit run dashboard.py

## Struktur Folder
- data/ → dataset mentah
- dashboard/ → dashboard
- notebook.ipynb → analisis
